# Алгоритм: как заполнять таблицу ссылками и ценами товаров

## Цель
Для каждого товара найти релевантную карточку на каждом целевом сайте и записать:
- ссылку на карточку
- цену
- валюту

## Рабочий pipeline

1. **Прочитать структуру Excel**
   - Определить колонку с названием товара.
   - Определить пары колонок `Ссылка / Цена / Валюта` по каждому сайту.

2. **Поиск кандидатов ссылок**
   - Делать поиск по запросу: `"<название товара> site:<домен>"`.
   - Брать только URL, которые выглядят как карточка товара (а не категория/тег/подборка).

3. **Проверка релевантности карточки**
   - Проверить, что на карточке есть совпадение по модели/артикулу.
   - Проверить, что товар в наличии (если это явно видно на странице).

4. **Извлечение цены**
   - Извлечь цену именно с этой карточки.
   - Взять валюту из карточки или по символу валюты.
   - Если цена не извлеклась уверенно — оставить пусто и пометить как `price n/a`.

5. **Запись в таблицу**
   - Записывать ссылку/цену/валюту в соответствующие колонки сайта.
   - Не смешивать сайты и не писать цену без ссылки.

6. **Контроль качества**
   - Проверять пары "ссылка+цена" (обе должны быть заполнены).
   - Отбрасывать очевидно нерелевантные ссылки (категории, теги, листинги без конкретного товара).

## Практика по инструментам
- Поиск ссылок: web search / Firecrawl search.
- Извлечение цены: Firecrawl scrape / extract по найденной карточке.
- Формирование результата: Excel (раздельные столбцы для ссылки, цены, валюты).

## Что считать успехом
- По каждой заполненной ячейке цены есть кликабельная ссылка на карточку товара.
- При ручной проверке по ссылке открывается тот же товар и цена соответствует записанной.
